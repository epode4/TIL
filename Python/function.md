# Function

## 함수 선언 & 호출

### 함수 선언
```python 
def function_name(parameter1, parameter2, ...):
    code1
    code2
    ...
    return value
```

### 함수 호출
```python
function_name(parameter1, parameter2, ...)
```

<br>

### Return

함수가 return을 만나면 해당 값을 반환하고 함수 종료
- 만약 return이 없다면 None 자동 반환
- 하나의 객체만 반환(여러 개 반환 원하면 tuple 형태로 만들기)
