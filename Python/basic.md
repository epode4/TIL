# Python_jupyter notebook 사용법

## 단축키

- ctrl + enter : 현재 cell 실행
- shift + enter : 현재 cell 실행 + 다음 cell로 이동
- alt + enter : 현재 cell 실행 + 새로운 cell 추가
- enter : insert 상태 실행
- esc : insert 상태에서 벗어나기
    - m : markdown으로 변환
    - y : code로 변환
    - x : cell 삭제 

## 주의사항

1. 대소문자 구분
